# Virtual Art Gallary
This App was Created with Reactjs and Nodejs with Postgres database.

## Available Scripts

### Client 
```
cd client
npm install
npm start
```

### Server 
```
cd server
npm install
npm run start
```

## Deployment
This website is deployed on Render - [Link](https://spur-challenge.onrender.com/)
